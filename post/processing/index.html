介绍

Processing 是一门画画的编程语言，它让设计师有了全新的表达方式，写几行代码就能创造出惊艳的视觉效果。Processing 更是可以和各种各样的软硬件交互，制作互动投影，音画联觉等新媒体内容。

Processing 可以用三个标签来总结：编程，视觉，易学

Processing 语言

Processing 是用 Java 编程语言写的，并且 Java 语言也是在语言树中最接近 Processing 的。所以，如果您熟悉 C 或 Java 语言，Processing 将很容易学。并且在程序如何构造方面，也作了一些简化。Processing 并不包括 Java 语言的一些较为高级的特性，但这些特性中的很多特性均已集成到了 Processing， 所以您无需了解它们。

void setup() // 程序运行的初始化函数，只会执行一次
void draw() // // 绘画，画完后显示出来，这个函数在程序运行期间会周期性执行
color c = color(200); // 定义颜色的一个类
//color(20); // 灰度
//color(0, 0); // 灰度，透明的
//color(255, 0, 0); // RGB
//color(255, 0, 0, 1); // RGB，透明度
size(200, 200); // 画布大小，width,height
background(0, 0); // 背景颜色，参数 color
fill(255, 0, 0); // 颜色填充，参数 color
noFill(); // 去除颜色填充
strokeWeight(10);// 构造图形线框粗细；
stroke(200);// 构造图形线框颜色，参数 color
noStroke(); // 去除线框
ellipse(300, 400, 200, 200); // 椭圆，圆心X轴距离，圆心Y轴距离，X轴宽度，Y轴宽度
rect(200, 200, 200, 200); // 矩形，左边X轴距离，上边Y轴距离，X轴宽度，Y轴宽度
println(random(0, 600)); // 取随机值
frameRate(20); // 控制 draw 每秒运行的次数，可以理解为帧数
